# Integração de APIs com React

Esta aplicação React demonstra a integração com a API JSONPlaceholder para buscar e exibir usuários.

## Funcionalidades

- Busca automática de usuários ao carregar a página
- Botão para buscar usuários manualmente
- Indicador de carregamento
- Tratamento de erros
- Filtro em tempo real por nome

## Como executar

1. Instale as dependências: `npm install`
2. Execute o servidor de desenvolvimento: `npm run dev`
3. Abra http://localhost:5173 no navegador

## Tecnologias

- React
- Vite
- Axios
